# Seaborn

Seaborn ([Seaborn homepage](https://seaborn.pydata.org/))
is a Python visualisation library based on matplotlib.
It provides a high-level interface for drawing attractive statistical graphics.

## Loading a Seaborn module

Here is how to load a Seaborn module:

```bash
module load Seaborn/0.13.2-gfbf-2024a
```

## Finding the Seaborn modules

Here is how to find the Seaborn modules:

```bash
module spider seaborn
```
